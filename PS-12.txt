1. Log In to AWS Console
2. Click on EC 2
3. Click on Launch EC 2 instance
4. Give Name
5. Let the Rest of the Settings as it is (Storage, AMI)
6. Click on Key Pair -> Create Key Pair
7. Give a Name e.g - "moba-ex"
8. Select .pem format for Moba Xterm
9. Click on Create Key Pair (It will autoamtically download the key (Preserve it in safe Folder))
10. Click on Create Instance

11. Now Open Moba Xterm
12. Click on "Start Local Terminal"
13. Navigate to the Directory where the key is downloaded

13. Go to Instance Dashboard 
14. Select the Created Instance -> Click on Connect
16. Navigate to SSH Client 
17. Copy the 3rd Command - e.g - chmod 400 "moba-ex.pem"
18. Paste it on moba xterm
19. Copy the ex Command - e.g - ssh -i "moba-ex.pem" --ec2--url
20. Paste in on moba xterm

And All Set ..!
